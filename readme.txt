== Description ==

Tyrone turns a WordPress installation into a website monitoring tool. Check the status of your sites, and keep tabs on which need upgrading, and scan for spam.

Future plans include cron jobs for automated scans on a recurring basis.

== Installation ==

1.Use automatic installer
1.Look for new menu item **Sites**
1.Add sites you wish to monitor, or import using the **Import Sites** menu option
1.Site will be scanned once initially when you publish
1.Use the **Run Tyrone Prowl** menu option to scan all your sites
